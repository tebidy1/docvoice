# أوامر التشغيل (Run Commands)

إليك الأوامر اللازمة لتشغيل التطبيقات الثلاثة يدوياً.

## 1. تطبيق سطح المكتب (Desktop App - Windows)
هذا هو التطبيق الرئيسي الذي يعمل كأداة (Widget) على نظام ويندوز.

```powershell
cd e:\d\DOCVOICE-ORG\docvoice
flutter run -d windows
```

## 2. إضافة المتصفح / الويب (Chrome Extension / Web)
لتشغيل نسخة الويب أو اختبار الإضافة على متصفح كروم.

```powershell
cd e:\d\DOCVOICE-ORG\docvoice
flutter run -d chrome
```

## 3. تطبيق الهاتف (Mobile App - Android/iOS)
التطبيق المرافق للهواتف الذكية. يقع في مجلد فرعي منفصل.

```powershell
cd e:\d\DOCVOICE-ORG\docvoice\mobile_app
flutter run
```
